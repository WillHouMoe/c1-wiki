# 梗指南

项目负责人：hcq</br>
参与撰稿人（按拼音首字母排序）：cjs、hcq、jcj、szh、xmy


梗是初中四年的一个缩影，它们或承载着一个时代，一段往事，抑或是一幅情景，一个人……

我们收录了其中的一百余条，给每个梗作传，介绍它的“前世今生”，作为永恒的纪念。

欢迎同学们随时随地打开本网站，翻开尘封的记忆，回味初中四年的美好时光。
