# 为考上华二蓄力

## 来源

事情起源于一节数学课。昨晚与学业鏖战了一晚上的 dyt 同学在下课时养精蓄锐，但是他似乎没有注意到上课铃声。这时，刘总走了进来……

刘总先试图使用他那标志性的清嗓声、翻书声来唤醒 dyt，但均以失败告终。见此情景，一向十分严肃的刘总也蚌埠住了，笑道：“dyt 同学是在为考上华二蓄力”。

dyt 同学的志向之远大，被同学们称道。

## 具体使用

一般地，“当 dyt 在睡觉”成立时，我们就可以说出这句话（不分是否是课堂）。

这句话的忠实使用者是 wxn。
