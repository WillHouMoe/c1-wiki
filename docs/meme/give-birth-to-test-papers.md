# 生小卷子

!!! warning "叠甲"
    
    本文不带有冒犯意味，请勿随意借本文内容调侃当事人或其他人。

## 来历

事情起源于一节英语课。当时 C 君正要评讲一张周测卷，不料有近一半的同学都忘带了（其实此时距离做这张卷子已经过了将近 $2$ 周，大多同学都已经忘记了它的存在）

C 君先是进行了一系列动作制造巨大的动静来威慑气氛，然后双手叉腰，嚷道：“你们有没有做学生的样子？不带试卷，干嘛啦？放在家里生小卷子啦？”

## 具体使用

主要是 C 君自己使用，通常用于询问同学为何不带卷子。

此正是：
<pre>
一周何度英语课，C 君进班大家坐。
此刻我欲评试题，C 君目光风欲雨。
不料台下无人“吱”，C 君破防正当时。
尔等实无同学之耻！吾班何能如此！
我欲评卷汝弗携，不知有颜来上学？
莫不居家生小卷，使我不得开心颜？
</pre>
